# Git Copy Paste Guide

## First Time Only
```bash
git checkout -b [yourname] && git push -u origin [yourname]
```

---

## Morning
```bash
git checkout main && git pull origin main && git checkout [yourname] && git merge main
```

## Before Going Home
```bash
git add . && git commit -m "work done" && git push
```
